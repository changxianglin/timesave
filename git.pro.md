### 1) git 练习

#### 1.1 ) atom 预览 markdown 语法的结果命令
```
ctr + shift + p 打开命令输入框
输入 mdpt(markdown preview toggle) 后可以在左边写右边查看结果
```

#### 1.2 ) atom 快捷键查看 mardown 预览 ctr + shift + M
